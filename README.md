# 📱 Simulado · Prova de Concurso

Gerador de simulado PWA (Progressive Web App) para prova de concurso com distribuição aleatória de questões por tópico.

## ✦ Como publicar no GitHub Pages

### 1. Crie o repositório
- Acesse [github.com](https://github.com) e faça login
- Clique em **"New repository"**
- Dê um nome (ex: `simulado-concurso`)
- Deixe **público** e clique em **"Create repository"**

### 2. Faça upload dos arquivos
- Na página do repositório, clique em **"Add file → Upload files"**
- Arraste **todos** os arquivos e a pasta `icons/` para a área de upload
- Clique em **"Commit changes"**

### 3. Ative o GitHub Pages
- Vá em **Settings → Pages**
- Em "Branch", selecione **main** e pasta **/ (root)**
- Clique em **Save**
- Aguarde ~1 minuto e seu app estará em:
  ```
  https://seu-usuario.github.io/simulado-concurso
  ```

---

## 📲 Como instalar no celular

### Android (Chrome)
1. Acesse o link do GitHub Pages
2. Um banner aparecerá automaticamente: toque em **"Instalar"**
3. O app aparecerá na tela inicial como um app nativo

### iPhone (Safari)
1. Acesse o link no **Safari** (não funciona em outros navegadores no iOS)
2. Toque no ícone de **Compartilhar** (↑)
3. Role e toque em **"Adicionar à Tela de Início"**
4. Toque em **Adicionar**

## 📋 Especificações do simulado

| Matéria     | Questões | Tópicos        |
|-------------|----------|----------------|
| Português   | 10       | POR 1 a POR 6  |
| Legislação  | 15       | LEG 1 a LEG 6  |
| Enfermagem  | 35       | ENF 1 a ENF 22 |
| **Total**   | **60**   |                |

**Regras de distribuição:**
- Mínimo de 1 questão por tópico garantido
- O restante é distribuído aleatoriamente entre os tópicos
- A ordem das questões é embaralhada
- Funciona offline após a primeira visita
